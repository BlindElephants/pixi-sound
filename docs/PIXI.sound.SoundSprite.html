Class: SoundSprite

PIXI.sound.SoundSprite

new PIXI.sound.SoundSprite (parent, options)

Object that represents a single Sound's sprite.

Name Type Description
parent PIXI.sound.Sound

The parent sound
options Object

Data associated with object.

Name Type Description
start number

The start time in seconds.
end number

The end time in seconds.
speed number optional

The optional speed, if not speed, uses the default speed of the parent.
